# LinkedIn Organic Content Strategy – B2B SaaS

## Overview
This project analyzes LinkedIn content strategies used by top B2B SaaS operators.

The goal was to identify patterns across high-performing creators and understand how content drives growth, trust, and pipeline.

## Data Collected
- 30 LinkedIn posts (3 per expert)  
- 6 YouTube videos (insight extraction)

---

## Key Insights
- Hooks drive attention  
- Systems enable consistency  
- Distribution > creation  
- Value-first builds trust  
- Content drives pipeline indirectly
